# Logo Concept Explorations

These are exploration drafts, not the final production brand.

## Directions

- `logo-concept-a-loop-monogram.svg`: open loop + monogram, closest to a premium app icon
- `logo-concept-b-prompt-gate.svg`: terminal/product tooling direction with the clearest developer cue
- `logo-concept-c-linked-orbit.svg`: multi-agent / orchestration direction with a collaborative feel
- `logo-concept-d-solid-aperture.svg`: strongest geometric silhouette, closest to infrastructure-brand minimalism
