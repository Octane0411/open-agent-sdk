---
title: ReAct Loop
description: Understand the multi-turn reasoning and tool execution loop.
---

Open Agent SDK uses a ReAct-style loop:

1. Model reasons about the current objective.
2. Agent executes tool calls when needed.
3. Tool results are fed back to the model.
4. Loop continues until completion or limit.

This design enables robust multi-turn behavior for file, shell, web, and task workflows.
