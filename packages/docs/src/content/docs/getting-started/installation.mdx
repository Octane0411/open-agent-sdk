---
title: Installation
description: Install Open Agent SDK and prepare your environment.
---

Install the SDK:

```bash
npm install open-agent-sdk@alpha
```

Alternative package managers:

```bash
yarn add open-agent-sdk@alpha
pnpm add open-agent-sdk@alpha
bun add open-agent-sdk@alpha
```

Requirements:

- Bun >= 1.0.0
- Node.js >= 18
- TypeScript >= 5.0
