Open Agent SDK is an open-source TypeScript SDK for building production agent workflows with Claude Agent SDK-style APIs, plus multi-provider support and extension points for real systems.

## What You Can Build

Use Open Agent SDK when you need:

- Stateful or one-shot agent workflows in TypeScript
- Built-in tools, permission controls, and session persistence
- Provider routing across OpenAI, Gemini, and Anthropic
- A migration-friendly path from Claude Agent SDK style integrations
