Open Agent SDK is a TypeScript SDK for building production-grade AI agents with tool use and multi-provider support.

Use this site for product-facing docs. Engineering workflows and ADRs remain in the repository root `docs/` directory.

## What you can do

- Build agents with a ReAct-style execution loop
- Connect OpenAI, Google Gemini, and Anthropic providers
- Register tools and control permissions
- Persist and resume sessions
